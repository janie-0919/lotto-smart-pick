번호 생성 통계 내 번호

통계 기반
로또 번호 추천

최근 회차 데이터를 분석해 편향 없는 균형 분산형 번호를 생성합니다

번호 생성 방식

  • 🎲 완전 랜덤 — 1~45 중 완전 무작위로 6개를 선택합니다.
  • 📊 통계 기반 — 최근 50회차 출현 빈도를 가중치로 적용해 번호를 선택합니다.
  • ⚖️ 균형형 — 구간 분산, 홀짝 비율, 합계 범위(100~175) 조건을 충족하는 번호를 선택합니다.

주요 기능

  • 5세트 동시 생성 및 번호 제외·포함 설정
  • HOT/COLD 번호 분석 — 최근 20회차 기준
  • 번호별 출현 빈도·홀짝·구간 분포 차트 (Recharts)
  • 생성한 번호 저장 및 최신 당첨 결과와 자동 비교

이용 방법

  1. 생성 방식(랜덤·통계·균형)을 선택합니다.
  2. 필요시 제외 또는 포함할 번호를 지정합니다.
  3. 번호 생성 버튼을 클릭해 5세트를 확인합니다.
  4. 마음에 드는 번호를 저장하고 당첨 결과와 비교해 보세요.
